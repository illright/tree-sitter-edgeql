This is a tree-sitter grammar project.

Never use complicated chains of terminal commands to accomplish a task. Never use `for` loops, `awk`, `sed`, heredocs with `EOF`, stuff like that. For creating and writing to files, use the built-in tools.

Don't write Perl/Python scripts to accomplish tasks like mass edits.

If you need to create temporary files, create them in the workspace using VS Code's built-in tools and then delete them when you're done. This way, you avoid having to request permissions and you can work uninterrupted.
